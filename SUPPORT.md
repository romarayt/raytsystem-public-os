# Support

raytsystem is a pre-1.0 open-source project maintained on a best-effort basis.

## Where to ask

- Setup, usage, and troubleshooting: open a GitHub Discussion after Discussions are enabled, or use
  the documentation issue form.
- Reproducible bugs: use the bug report form and include `raytsystem doctor` output after removing
  paths, tokens, corpus content, and other private data.
- Feature proposals: use the feature request form and describe the user problem before a solution.
- Security vulnerabilities: follow [SECURITY.md](SECURITY.md) and report privately. Never include a
  vulnerability or secret in a public issue.

There is no paid support SLA. A response time is not guaranteed.
